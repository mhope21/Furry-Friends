# Furry Friends Website

Welcome to our mock dog adoption website! This website allows users to explore information about different dog breeds and find their perfect furry companion.

## Features

- **Browse Dog Breeds**: Browse through a list of dog breeds to learn more about their characteristics and personalities.
- **View Dog Images**: See images of each dog breed to get a visual representation of their appearance.
- **Learn About Temperament**: Discover the temperament and personality traits associated with each breed.


## How to Use

1. **Explore Breeds**: Navigate through the list of dog breeds to find one that interests you.
2. **View Details**: Click on a breed to view more details, including an image, temperament, and recommended activities.
3. **Consider Adoption**: If you find a breed that you love, consider adopting from us! Information about how to get in touch with us is provided.

## Technologies Used

- HTML
- CSS
- JavaScript

## Usage

To use this website, simply open the `index.html` file in a web browser.
